1

대상군

만 4–6세 IDD(발달장애) 아동 중 소근육(fine motor) 저하가 확인된 아동 (조기개입 연령대)

포함 진단군 (예시)

CP · 뇌성마비ASD · 자폐 Down syndromeDCD기타 IDD

⚠️ 진단명이 아니라
손기능 저하 여부가 핵심 기준.

2

포함 / 제외 기준

포함 기준

  • 만 4–6세
  • IDD / 발달장애 진단 (DSM-5 / ICD-11)
  • 소근육(fine motor) 저하 — 표준검사 ≤16%ile
  • 간단한 지시 이해 가능
  • 보호자 도움 하 1일 400회 능동반복 수행 가능
  • 가정 내 주 5회 이상 사용 가능

제외 기준

  • 조절되지 않는 발작
  • 심한 시각 / 청각 문제
  • 중증 행동문제 (게임 수행 방해)
  • 최근 보톡스 / 수술 / 집중 상지치료
  • 평가 불가능한 인지 / 행동 수준

진단 구성 — 모집 기반
포함 게이트 = IDD + 소근육 저하 (CP·ASD·DCD·다운 등). 진단명은 기록·층화 + 진단×중재 상호작용 사전분석. 진단 구성은 각 기관 모집 가능성에 따르되 특정 진단 편중 방지·주요 진단군 최소 N 확보. 검정력은 completer 기준이라 탈락 15–20% 대비 등록 버퍼 둘 것. 효과크기는 파일럿(세브란스 N=27)으로 재산출.

The key is not “all IDD” but “children with developmental disorders who have manual dexterity impairment.”
Broaden the diagnoses, but fix the functional problem to a single criterion.

Inclusion / Exclusion

Inclusion

  • Age 4–6 years
  • IDD / developmental diagnosis (DSM-5 / ICD-11)
  • Low fine motor — ≤16th %ile on a standardized test
  • Understands simple instructions
  • Can complete ~400 active reps/day with caregiver
  • Home use ≥ 5×/week feasible

Exclusion

  • Uncontrolled seizures
  • Severe visual / auditory impairment
  • Severe behavioral issues (block gameplay)
  • Recent botox / surgery / intensive UL therapy
  • Cognitive/behavioral level precluding assessment

Diagnosis composition — recruitment-driven
